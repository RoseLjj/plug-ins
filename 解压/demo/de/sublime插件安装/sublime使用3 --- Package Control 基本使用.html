呼出Package Control

要使用Package Control首先需要将其呼出,有如下两种方法.

方法一:快捷键

  1. ctrl+shift+p呼出界面
  2. 输入package control在出现的选项中选取功能

方法二:菜单选取

  1. 菜单选择: preference(首选项) -> Package control
  2. 在弹出的界面中选择功能

安装插件

  1. 选取 install Packages
  2. 在弹出的对话框中,输入插件名,回车等待安装即可
  3. 关于常用的插件,请看后续的文章

删除插件

  1. 选取 remove Packages
  2. 在弹出的对话框中,选择插件名,回车等待删除即可
  3. 在sublime的左下角,能够看到进度信息

查询已安装插件

  1. 选取 list Packages
  2. 在弹出的对话框中,即可查看已安装的插件
