Package Control概述

为了能够方便的管理繁多的插件,一般我们会先安装Package Control这个插件

Package Control 官网首页

让我们看看首页的描述

  1. The Sublime Text package manager that makes it exceedingly simple to find, install and keep packages up-to-date.
  3. Sublime Text的包(插件)管理工具,让(插件的)查找、安装、更新 等工作都变得极为简单

安装

  • 进入安装页面
    点击 Installation 选项

安装方法一 Simple部分:

让我们看看教程

  1.     The simplest method of installation is through the Sublime Text console. The console is accessed via the ctrl+` shortcut or the View > Show Console menu. Once open, paste the appropriate Python code for your version of Sublime Text into the console.
  3.     最简单的方法是通过Sublime Text的控制台.控制台的呼出方式有: 使用 ctrl+` 快捷键 或者在 菜单栏中选择 View -> Show Console menu.打开以后,复制对应Sublime版本的Python代码到控制台中 回车 等待即可
  5.     需要注意的是,不要复制来路不明的Python代码,直接使用官网的,避免出现不必要的麻烦

安装方法二 Manual部分

同样先看看教程

  • 说明

    1. If for some reason the console installation instructions do not work for you (such as having a proxy on your network), perform the following steps to manually install Package Control:
    3. 如果由于某些原因,使用控制台无法安装成功(比如网络问题),可以使用下列步骤进行手动安装

  • 步骤

    1. 1.Click the Preferences > Browse Packages… menu
    2. 2.Browse up a folder and then into the Installed Packages/ folder
    3. 3.Download Package Control.sublime-package and copy it into the Installed Packages/ directory
    4. 4.Restart Sublime Text
    6. 1. 点击菜单栏中的 Preference(首选项) > Browse Packages(浏览插件) 选项
    7. 2. 往上跳一级文件夹,然后进入 InstallEd Packages 文件夹下
    8. 3. 下载 Package Control.sublime-package 然后拷贝到 Installed packages 目录下
    9. 4. 重启 Sublime Text

    点击下载 Package Control.sublime-package
