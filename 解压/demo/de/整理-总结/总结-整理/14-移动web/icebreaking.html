开放性讨论

怎么去学习新的技术？

  • 官网文档先把文档过一遍
    • 版本信息，文章的发布时间
    • 混个眼熟，知道api有哪些，这个新技术可以做什么事情就可以。
    • 一句话就能概括一个技术的，不要想的太复杂
  • 上手写demo
    • 一定要亲手取用每一个api
  • 去找一些第三方的资源
    • 去看github上的开源的项目
    • 找一些大牛的博客
    • 书
    • 汇智网，慕课网
  • 总结自己学过的知识
    • 画脑图
    • 写博客
      • 开源精神
  • 先what然后how最后why

  • 安装npm install browser-sync -g
