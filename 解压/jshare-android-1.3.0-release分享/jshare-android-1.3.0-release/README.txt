JShare SDK for Android 简介
===========================
JShare SDK for Android 是一套由极光提供的软件库，使开发者可将极光的分享服务集成到自己的 Android app 中。

关于 JShare SDK for Android 的集成，请查阅我们详细的集成指南和 API 指南。
也可参考下载的压缩包中附带的示例 demo 来通过代码学习本 SDK 的用法。


新手指南
--------
0. 从官方路径（https://docs.jiguang.cn/jshare/resources/）下载本 SDK，或者参考 JCenter 配置指南（https://docs.jiguang.cn/jshare/client/Android/android_sdk/#jcenter） 来配置 JCenter 的下载路径。
1. 复制 jar 文件、so 文件以及 /res 资源文件到你项目对应的文件夹下，配置 JShare 所必须的 AndroidManifest 组件（https://docs.jiguang.cn/jshare/client/Android/android_sdk/#_4）；如果是 JCenter 集成方式会自动完成这一步。
2. 配置 JGShareSDK.xml，无论是使用自动集成还是手动集成方式，都需要配置 JGShareSDK.xml。配置指南（https://docs.jiguang.cn/jshare/client/Android/android_sdk/#jgsharesdkxml）
3. 配置微信平台回调，无论是使用自动集成还是手动集成方式，都需要配置，配置指南（https://docs.jiguang.cn/jshare/client/Android/android_sdk/#_5）
4. 配置项目签名，无论是使用自动集成还是手动集成方式，都需要配置，配置指南（https://docs.jiguang.cn/jshare/client/Android/android_sdk/#_6）
5. 基础 api 教程见 demo 实例，详细 api 解释见：https://docs.jiguang.cn/jshare/client/Android/android_api/
